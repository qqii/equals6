# equals6
Figuring out the =6 problem through brute force and javascript

## Problem

Given n (- {2..9}, m = 6 find `f :: Number -> Number -> Number`, `g :: Number -> Number -> Number` such that:

`g(f(n, n), n) = m`

where Number is ill defined. 

### Commutative

V- commutative f, g:

`g(f(n, n), n) = m = f(g(n, n), n))`
